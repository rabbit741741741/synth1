Publicar esta PWA no GitHub Pages
=================================

Objetivo: obter um link HTTPS do tipo:
https://SEU_UTILIZADOR.github.io/synthlab-isochronic/

Passos simples
--------------
1. Crie uma conta GitHub se ainda não tiver.
2. Crie um novo repositório chamado, por exemplo:
   synthlab-isochronic
3. Faça upload de TODOS os ficheiros desta pasta:
   index.html
   style.css
   app.js
   presets.js
   manifest.json
   service-worker.js
   icons/
4. No GitHub, abra:
   Settings → Pages
5. Em "Build and deployment", escolha:
   Source: Deploy from a branch
   Branch: main
   Folder: /root
6. Aguarde alguns minutos.
7. Abra o link HTTPS fornecido pelo GitHub Pages no Chrome do Android.
8. Menu ⋮ → Adicionar ao ecrã principal.

Notas
-----
- O GitHub Pages é apenas uma forma simples de alojar ficheiros estáticos.
- Esta PWA não precisa de servidor próprio nem base de dados.
- Para atualizar a app, substitua os ficheiros no repositório.
