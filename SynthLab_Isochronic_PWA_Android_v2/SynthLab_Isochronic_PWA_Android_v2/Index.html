Nota sobre os presets

Os presets Solfeggio/isocrónicos são mapeamentos simbólicos, subjetivos e experimentais. A aplicação gera áudio para exploração sonora, relaxamento e meditação. Não é um dispositivo médico, não diagnostica, não trata, não cura e não previne doenças. Use volume baixo e interrompa a utilização se sentir desconforto.

Como instalar no Android
  1. Aloje estes ficheiros num endereço HTTPS, por exemplo GitHub Pages.
  2. Abra o link no Chrome do Android.
  3. Menu ⋮ → Adicionar ao ecrã principal.
  4. Depois de abrir uma vez, a app pode funcionar offline.

Abrir só o ficheiro index.html pode tocar som, mas normalmente não ativa PWA/offline completo.
